# Images and Supporting Materials

This folder contains images, screenshots, and diagrams used in the Building Genie Agents lab.

## Contents

This folder is intended for:
- **Screenshots** of the Genie UI and space configuration
- **Diagrams** illustrating Genie architecture and workflows
- **Visual guides** for navigating Databricks Genie features
- **Example outputs** showing Genie responses and visualizations

## Usage

Images in this folder are referenced in the lab notebooks to provide visual context and step-by-step guidance for learners.

## Contributing

When adding images:
1. Use descriptive filenames (e.g., `genie_space_configuration.png`)
2. Keep file sizes reasonable (< 1MB per image)
3. Use PNG format for screenshots and diagrams
4. Update this README with descriptions of new images
