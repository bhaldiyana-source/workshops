## Adding Images

When adding images to this directory:

1. **Use descriptive filenames:**
   - `salesforce-connected-app-setup.png`
   - `oauth-flow-diagram.png`
   - `customer-360-dashboard.png`

2. **Reference in documentation:**
   ```markdown
   ![Salesforce Connected App Setup](images/salesforce-connected-app-setup.png)
   ```

3. **Keep file sizes reasonable:**
   - Use PNG for screenshots
   - Use SVG for diagrams when possible
   - Compress images before adding to repository

4. **Organize by category (optional):**
   - `setup/` - Setup and configuration screenshots
   - `diagrams/` - Architecture and flow diagrams
   - `dashboards/` - Analytics and visualization examples
