# Images and Supporting Materials

This folder contains supporting images, diagrams, and visual materials for the Building Agent Tools on Databricks lab.

## Contents

You can add the following types of files here:

### Architecture Diagrams
- Agent execution flow diagrams
- Unity Catalog function architecture
- Tool calling sequence diagrams

### Screenshots
- AI Playground interface examples
- Function creation steps
- Testing and debugging views

### Conceptual Diagrams
- Agent decision-making flows
- Function design patterns
- Best practices illustrations

### Example Visuals
- Sample data models
- UC namespace structures
- Permission and governance diagrams

## File Naming Convention

Use descriptive names with prefixes:
- `arch-*.png` - Architecture diagrams
- `screenshot-*.png` - UI screenshots
- `diagram-*.png` - Conceptual diagrams
- `example-*.png` - Example visuals

## Usage in Notebooks

Reference images in notebooks using markdown:

```markdown
![Description](../images/filename.png)
```

Or using HTML for more control:

```html
<img src="../images/filename.png" alt="Description" width="600"/>
```
